# 변경 기록

이 프로젝트의 주요 변경 사항을 버전별로 기록합니다.

## [1.0.1] - 2026-05-27

### 변경

- 일기 작성, 목록, 상세, 달력, 검색, 설정 화면의 프론트엔드 구성을 정리했습니다.
- 하단 내비게이션을 추가해 주요 화면 간 이동을 단순화했습니다.
- 달력에서 일기 개수 대신 작성 여부를 배경색으로 표시하도록 변경했습니다.
- 일기 카드와 상세 화면에 날짜, 시간, 기분 메타 정보를 더 명확히 표시했습니다.
- 앱 테마 색상, 타이포그래피, 모서리 반경을 정리했습니다.
- Markdown 내보내기 진행 상태와 오류 처리를 보강했습니다.
- 앱 백업/기기 이전 데이터 추출을 명시적으로 차단했습니다.

### 추가

- Room DAO, Repository, Entity 검증 테스트를 추가했습니다.
- 기본 런처 아이콘과 다크 모드용 시스템 테마 리소스를 추가했습니다.
- README에 실행 방법을 보강했습니다.

### 수정

- 에디터의 미저장 변경 감지가 날짜, 시간, 기분 변경까지 포함하도록 수정했습니다.
- 기분 코드 검증과 빈 기분 값 정규화를 보강했습니다.
- `entryTimeMinute` 범위 검증을 보강했습니다.
